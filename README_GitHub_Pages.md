# ESP32 Umweltmessstation - interaktive Website

## Verwendung auf GitHub Pages

1. Den Inhalt dieses ZIP-Pakets in das Repository hochladen.
   - `index.html` muss im Root-Verzeichnis liegen.
   - Der Ordner `assets/` muss daneben liegen.
2. In GitHub auf **Settings > Pages** gehen.
3. Unter **Build and deployment** einstellen:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. Speichern und ein paar Minuten warten.

## Wichtige Dateien

- `index.html` - interaktive Projektwebsite.
- `assets/ESP32_Umweltstation_Projektdokumentation_FINAL_v2.pdf` - finale PDF-Doku.
- `assets/esp32_umweltstation_final.ino` - finaler Arduino-Code.
- `assets/*.png` und `assets/*.jpg` - Bilder und Screenshots.

## Anpassung

Wenn du neue Fotos machst, lege sie in `assets/` und ersetze die passenden Bildpfade im Abschnitt `Bilder und Nachweise`.
