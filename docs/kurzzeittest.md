# Kurzzeittest

Ein vollständiger 24h Dauertest konnte vor der Präsentation nicht durchgeführt werden.

Stattdessen wurde ein mehrstündiger Test durchgeführt.

## Geprüfte Funktionen

- Display
- Touch
- RTC
- BME680
- Statistik
- Graphanzeige
- Alarmprotokoll
- Buzzer
- Button

## Ergebnis

Alle Kernfunktionen arbeiteten stabil.

Die Sensorwerte wurden erfolgreich mit einem externen Hygrometer verglichen.

Die Luftqualitätsbewertung wurde durch verschiedene Tests überprüft:

- Fenster offen
- Fenster geschlossen
- Angepusteter Sensor
