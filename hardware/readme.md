# Hardware

Dieser Ordner enthält alle hardwarebezogenen Informationen der Umweltmessstation.

## Inhalte

### Bauteilliste.md

Übersicht aller verwendeten Komponenten.

### Verkabelung.md

Dokumentation der Pinbelegung und Verdrahtung zwischen den einzelnen Komponenten.

### Schaltplan.png

Grafische Darstellung der elektrischen Verschaltung des Systems.

## Verwendete Hauptkomponenten

- ESP32 Dev Board
- BME680 Sensor
- DS3231 RTC Modul
- ST7796S Display
- XPT2046 Touchcontroller
- Aktiver Buzzer
- Push Button
- 3D-gedrucktes Gehäuse
