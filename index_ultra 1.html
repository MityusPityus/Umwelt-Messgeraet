Projektkern

1. Einleitung

Entwickelt wurde eine kompakte Umweltmessstation auf ESP32-Basis. Das System erfasst Raumwerte, verarbeitet sie lokal und zeigt sie auf einem 4-Zoll-TFT-Touchdisplay an. Im Mittelpunkt stehen Temperatur, Luftfeuchtigkeit, Luftdruck und die qualitative Luftbewertung über den Gaswiderstand des BME680.

Das fertige System arbeitet offline: keine Cloud, keine API, kein WLAN. Die Zeit kommt von einem DS3231 RTC-Modul.

2. Projektplanung

Die Umsetzung erfolgte iterativ. Einzelkomponenten wurden separat getestet und anschließend in die Hauptsoftware integriert. Das GitHub Repository trennt docs, hardware, software, tests und 3D-Modelle. Das Scrumboard enthielt Backlog, To Do, In Progress, Testing, In Review und Done.

3. Architektur

Der ESP32 verbindet BME680, DS3231, ST7796S-Display, XPT2046-Touch, Button, Buzzer und lokale LittleFS-Speicherung. Dadurch entsteht ein lokales Cyber-Physical System.

Finale Erweiterungen

Touch, Statistik, Graph

Im finalen Stand kamen Touch-Navigation, Statistikscreen und ein 24h-Graph hinzu. Für die Präsentation wurde der Zeitraum von 14 Tagen auf 24 Stunden geändert, damit schnell sichtbare Daten entstehen.

Alarmprotokoll

1. Einleitung
1.1 Projektbeschreibung
Im Rahmen des Projekts wurde eine kompakte Umweltmessstation auf Basis eines ESP32 entwickelt. Das
System erfasst Umgebungswerte in einem Raum, verarbeitet sie lokal und zeigt sie auf einem 4-Zoll-TFTTouchdisplay an. Im Mittelpunkt stehen Temperatur, Luftfeuchtigkeit, Luftdruck und eine qualitative Bewertung
der Luftqualität anhand des Gaswiderstands des BME680 Sensors.
Das fertige System arbeitet offline. Es nutzt keine Cloud, keine externe API und keine WLAN-Verbindung.
Datum und Uhrzeit werden über ein DS3231 RTC-Modul bereitgestellt. Zusätzlich besitzt das System eine
akustische Alarmfunktion, eine Alarmquittierung über einen Taster, ein Touchmenü, eine lokale
Messwertspeicherung, eine Statistikansicht, einen 24h-Graph und ein Alarmprotokoll.

1.2 Ausgangssituation und Ist-Zustand
Zu Projektbeginn lagen keine fertigen Komponenten und kein aufgebautes Gesamtsystem vor. Zunächst
mussten geeignete Hardwarekomponenten ausgewählt, getestet und auf Kompatibilität geprüft werden. Dabei
mussten mehrere Schnittstellen kombiniert werden: SPI für Display und Touch, I2C für BME680 und RTC sowie
digitale GPIOs für Button und Buzzer.
Im finalen Stand sind alle Kernkomponenten integriert. Das defekte erste RTC-Modul wurde ersetzt, Display und
Touch wurden kalibriert, das Gehäuse wurde gedruckt und montiert, Sensorwerte werden angezeigt, Messwerte
werden lokal gespeichert und bei schlechter Luftqualität wird ein Alarm ausgelöst.

4. Umsetzung und Entwicklung
4.1 Entwicklungsumgebung
Als Entwicklungsumgebung wurde die Arduino IDE verwendet. Der serielle Monitor mit 115200 Baud war das
wichtigste Diagnosewerkzeug. Die Entwicklung erfolgte testgetrieben im praktischen Sinne: Jede wichtige
Komponente wurde zuerst einzeln getestet, bevor sie in die Hauptsoftware integriert wurde.

4.2 Display- und Touchintegration
Das ST7796S-Display wurde über SPI betrieben. Zunächst wurden Rotation, Farben, Textpositionen und
Ausrichtung getestet. Nach dem Einbau ins Gehäuse wurde das Layout kompakter gemacht, damit obere und
untere Inhalte nicht vom Gehäuserand verdeckt werden.
Der Touchscreen war ein wichtiger Debugging-Schritt. Die zuerst getestete Bonezegei-XPT2046-Bibliothek
zeigte im Zusammenspiel mit dem Display unzuverlässiges Verhalten. Die finale Lösung verwendet die
XPT2046_Touchscreen-Bibliothek. Danach wurden Achsen gespiegelt und ein kleiner X-Offset gesetzt, bis die
Touchposition exakt zur Anzeige passte.

gegen ein neues DS3231-Modul zählte die Zeit korrekt hoch. Damit war das RTC-Problem geloest.

4.5 Buzzer, Button und Alarm
Der Buzzer wird über GPIO26 betrieben und gibt bei schlechter Luftqualität ein periodisches Signal aus. Der
Button an GPIO13 verwendet INPUT_PULLUP und dient zur Quittierung. Wird die Luft wieder besser, wird die
Alarmquittierung automatisch zurückgesetzt.
Der Alarm wird ausgelöst, wenn der Gaswert unter 70 kOhm fällt. Beim Übergang von nicht schlecht zu schlecht
wird zusätzlich ein Alarmereignis mit Zeitstempel und Messwerten gespeichert.

4.6 Datenspeicherung, Statistik, Graph und Alarmprotokoll
Die Messwerte werden lokal über LittleFS im ESP32-Flash gespeichert. Es werden CSV-Zeilen mit Zeitstempel,
Temperatur, Luftfeuchtigkeit, Druck und Gaswert geschrieben. Für die Präsentation wurde das Speicherintervall
auf 30 Sekunden gesetzt, damit schnell sichtbare Daten für Statistik und Graph entstehen. Für den normalen
Betrieb kann das Intervall auf 5 Minuten gestellt werden.

9. Fazit, Reflexion und Ausblick
Das Projekt entwickelte sich von einer einfachen ESP32-Sensoridee zu einem vollständigen lokalen CyberPhysical System. Besonders erfolgreich war die Kombination aus Sensorik, Anzeige, Touchbedienung, lokaler
Speicherung und Alarmreaktion. Der finale Prototyp zeigt nicht nur Live-Werte, sondern erlaubt auch Statistik,
24h-Verlauf und Alarmhistorie.
Ein wichtiger technischer Lernpunkt war das systematische Debugging. Das RTC-Problem wurde nicht geraten,
sondern durch I2C-Scan, RTClib-Test, Registeranalyse und SQW-Test eingegrenzt. Auch beim Touchscreen
zeigte sich, dass eine passende Library entscheidend ist. Erst nach dem Wechsel auf XPT2046_Touchscreen
und nach Kalibrierung funktionierte die Bedienung sauber.
Besonders stolz bin ich auf das Display und die Darstellung der Sensorwerte. Der Hauptbildschirm wirkt wie ein
echtes kleines Messgerät, und die Zusatzscreens für Statistik, Graph und Alarmprotokoll machen das System
deutlich hochwertiger als einen reinen Sensortest.
Rückblickend habe ich gelernt, wie man ein technisches Projekt von Anfang bis Ende realisiert: Planung,
Hardwareauswahl, Tests, Fehlersuche, Softwareentwicklung, 3D-Gehäuse, Projektboard, Repository und
Dokumentation. Der nicht mehr durchgeführte 24h-Dauertest ist ein ehrlicher offener Punkt, aber die
Kernfunktionen wurden erfolgreich umgesetzt und im Kurzzeittest plausibilisiert.

if (epoch >= fromEpoch && epoch <= toEpoch) {
sTemp += temp;
sHum += hum;
sPressure += pressure;
sGas += gas;
result.count++;
}
}
}
file.close();
if (result.count > 0) {
result.hasData = true;
result.avgTemp = sTemp / result.count;
result.avgHum = sHum / result.count;
result.avgPressure = sPressure / result.count;
result.avgGas = sGas / result.count;
}
return result;
}

void calculateHourlyPoints(HourlyPoint points[24]) {


for (int i = 0; i < 24; i++) {
points[i].hasData = false;
points[i].hourNumber = 0;
points[i].avgTemp = 0;
points[i].avgHum = 0;
points[i].avgPressure = 0;
points[i].avgGas = 0;
}
DateTime now = rtc.now();
unsigned long currentHour = getHourNumber(now.unixtime());
float sumTemp[24];
float sumHum[24];
float sumPressure[24];
float sumGas[24];
unsigned long count[24];
for (int i = 0; i < 24; i++) {
sumTemp[i] = 0;
sumHum[i] = 0;
sumPressure[i] = 0;
sumGas[i] = 0;
count[i] = 0;
unsigned long hourNum = currentHour - (23 - i);
points[i].hourNumber = hourNum;
}
File file = LittleFS.open(LOG_FILE, FILE_READ);
if (!file) return;
while (file.available()) {
String line = file.readStringUntil('\n');
line.trim();
unsigned long epoch;
float temp, hum, pressure, gas;
if (parseLogLine(line, epoch, temp, hum, pressure, gas)) {
unsigned long hourNum = getHourNumber(epoch);
for (int i = 0; i < 24; i++) {
if (points[i].hourNumber == hourNum) {
sumTemp[i] += temp;
sumHum[i] += hum;
sumPressure[i] += pressure;
sumGas[i] += gas;
count[i]++;
break;
}
}
}
}
file.close();
for (int i = 0; i < 24; i++) {
if (count[i] > 0) {
points[i].hasData = true;
points[i].avgTemp = sumTemp[i] / count[i];
points[i].avgHum = sumHum[i] / count[i];
points[i].avgPressure = sumPressure[i] / count[i];
points[i].avgGas = sumGas[i] / count[i];
}
}
}

return;

lastBuzzerToggle = millis();
buzzerState = !buzzerState;
digitalWrite(BUZZER_PIN, buzzerState ? HIGH : LOW);
}
}

String getCsvField(String line, int index) {
  int currentIndex = 0;
  int start = 0;

  for (int i = 0; i <= line.length(); i++) {
    if (i == line.length() || line.charAt(i) == ';') {
      if (currentIndex == index) {
        return line.substring(start, i);
      }
      currentIndex++;
      start = i + 1;
    }
  }
  return "";
}

void pruneOldLogEntries() {
  DateTime now = rtc.now();
  unsigned long nowEpoch = now.unixtime();

  if (!LittleFS.exists(LOG_FILE)) return;

  File input = LittleFS.open(LOG_FILE, FILE_READ);
  File output = LittleFS.open("/temp_log.csv", FILE_WRITE);

  if (!input || !output) {
    Serial.println("Prune Messlog fehlgeschlagen.");
    if (input) input.close();
    if (output) output.close();
    return;
  }

  output.println("epoch;temp;hum;pressure;gas");

  while (input.available()) {
    String line = input.readStringUntil('\n');
    line.trim();

    unsigned long epoch;
    float temp, hum, pressure, gas;

    if (parseLogLine(line, epoch, temp, hum, pressure, gas)) {
      if (epoch <= nowEpoch && nowEpoch - epoch <= STORAGE_RETENTION_SECONDS) {
        output.println(line);
      }
    }
  }

  input.close();
  output.close();
  LittleFS.remove(LOG_FILE);
  LittleFS.rename("/temp_log.csv", LOG_FILE);
  Serial.println("Alte Mess-Logeintraege entfernt.");
}

  File file = LittleFS.open(LOG_FILE, FILE_READ);

  while (file.available()) {
    String line = file.readStringUntil('\n');
    line.trim();

    unsigned long epoch;
    float temp, hum, pressure, gas;

    if (parseLogLine(line, epoch, temp, hum, pressure, gas)) {
      if (epoch >= fromEpoch && epoch <= toEpoch) {
        sTemp += temp;
        sHum += hum;
        sPressure += pressure;
        sGas += gas;
        result.count++;
      }
    }
  }

  file.close();

  if (result.count > 0) {
    result.hasData = true;
    result.avgTemp = sTemp / result.count;
    result.avgHum = sHum / result.count;
    result.avgPressure = sPressure / result.count;
    result.avgGas = sGas / result.count;
  }

  return result;
}

AverageStats calculateTodayAverage() {
  DateTime now = rtc.now();
  DateTime startOfDay(now.year(), now.month(), now.day(), 0, 0, 0);
  return calculateAverageForRange(startOfDay.unixtime(), now.unixtime());
}

AverageStats calculateLast24HoursAverage() {
  DateTime now = rtc.now();
  unsigned long nowEpoch = now.unixtime();
  unsigned long fromEpoch = nowEpoch - LAST_24H_SECONDS;
  return calculateAverageForRange(fromEpoch, nowEpoch);
}

void calculateHourlyPoints(HourlyPoint points[24]) {
  for (int i = 0; i < 24; i++) {
    points[i].hasData = false;
    points[i].hourNumber = 0;
    points[i].avgTemp = 0;
    points[i].avgHum = 0;
    points[i].avgPressure = 0;
    points[i].avgGas = 0;
  }

  DateTime now = rtc.now();
  unsigned long currentHour = getHourNumber(now.unixtime());

  float sumTemp[24];
  float sumHum[24];
  float sumPressure[24];
  float sumGas[24];
  unsigned long count[24];

  for (int i = 0; i < 24; i++) {
    sumTemp[i] = 0;
    sumHum[i] = 0;
    sumPressure[i] = 0;
    sumGas[i] = 0;
    count[i] = 0;
    unsigned long hourNum = currentHour - (23 - i);
    points[i].hourNumber = hourNum;
  }

  File file = LittleFS.open(LOG_FILE, FILE_READ);
  if (!file) return;

  while (file.available()) {
    String line = file.readStringUntil('\n');
    line.trim();

    unsigned long epoch;
    float temp, hum, pressure, gas;

    if (parseLogLine(line, epoch, temp, hum, pressure, gas)) {
      unsigned long hourNum = getHourNumber(epoch);
      for (int i = 0; i < 24; i++) {
        if (points[i].hourNumber == hourNum) {
          sumTemp[i] += temp;
          sumHum[i] += hum;
          sumPressure[i] += pressure;
          sumGas[i] += gas;
          count[i]++;
          break;
        }
      }
    }
  }

  file.close();

  for (int i = 0; i < 24; i++) {
    if (count[i] > 0) {
      points[i].hasData = true;
      points[i].avgTemp = sumTemp[i] / count[i];
      points[i].avgHum = sumHum[i] / count[i];
      points[i].avgPressure = sumPressure[i] / count[i];
      points[i].avgGas = sumGas[i] / count[i];
    }
  }
}

void handleButton() {
  bool buttonPressed = (digitalRead(BUTTON_PIN) == LOW);
  if (buttonPressed && millis() - lastButtonPress > debounceDelay) {
    lastButtonPress = millis();
    if (airBad) {
      alarmMuted = !alarmMuted;
      if (alarmMuted) {
        Serial.println("Alarm quittiert.");
        digitalWrite(BUZZER_PIN, LOW);
        buzzerState = false;
      } else {
        Serial.println("Alarm wieder aktiviert.");
      }
      updateFooterStatus();
    } else {
      Serial.println("Button gedrueckt, aber kein Alarm aktiv.");
    }
  }
}

void handleBuzzer() {
  if (!airBad || alarmMuted) {
    digitalWrite(BUZZER_PIN, LOW);
    buzzerState = false;
    return;
  }

  if (millis() - lastBuzzerToggle >= buzzerInterval) {
    lastBuzzerToggle = millis();
    buzzerState = !buzzerState;
    digitalWrite(BUZZER_PIN, buzzerState ? HIGH : LOW);
  }
}

void handleTouch() {
  disableDisplay();
  if (!touch.touched()) return;
  if (millis() - lastTouchAction < touchDebounceDelay) return;

  TS_Point p = touch.getPoint();
  int screenX, screenY;
  mapTouchToScreen(p.x, p.y, screenX, screenY);

  Serial.print("Touch screen x=");
  Serial.print(screenX);
  Serial.print(" y=");
  Serial.print(screenY);
  Serial.print(" raw z=");
  Serial.println(p.z);

  if (currentScreen == SCREEN_MAIN) {
    if (isInside(screenX, screenY, STAT_BTN_X, STAT_BTN_Y, STAT_BTN_W, STAT_BTN_H)) {
      lastTouchAction = millis();
      currentScreen = SCREEN_STATS;
      statsNeedsRefresh = false;
      drawStatsScreen();
      return;
    }
  }

  if (currentScreen == SCREEN_STATS) {
    if (isInside(screenX, screenY, ALARM_BTN_X, ALARM_BTN_Y, ALARM_BTN_W, ALARM_BTN_H)) {
      lastTouchAction = millis();
      currentScreen = SCREEN_ALARMS;
      alarmLogNeedsRefresh = false;
      drawAlarmLogScreen();
      return;
    }

    if (isInside(screenX, screenY, GRAPH_BTN_X, GRAPH_BTN_Y, GRAPH_BTN_W, GRAPH_BTN_H)) {
      lastTouchAction = millis();
      currentScreen = SCREEN_GRAPH;
      graphNeedsRefresh = false;
      drawGraphScreen();
      return;
    }

    if (isInside(screenX, screenY, BACK_BTN_X, BACK_BTN_Y, BACK_BTN_W, BACK_BTN_H)) {
      lastTouchAction = millis();
      currentScreen = SCREEN_MAIN;
      drawStaticUI();
      updateClock();
      updateMainMeasurementDisplay();
      return;
    }
  }

  if (currentScreen == SCREEN_GRAPH) {
    if (isInside(screenX, screenY, GBTN_TEMP_X, GBTN_Y, GBTN_W, GBTN_H)) {
      lastTouchAction = millis();
      currentGraphMode = GRAPH_TEMP;
      graphNeedsRefresh = false;
      drawGraphScreen();
      return;
    }

    if (isInside(screenX, screenY, GBTN_HUM_X, GBTN_Y, GBTN_W, GBTN_H)) {
      lastTouchAction = millis();
      currentGraphMode = GRAPH_HUM;
      graphNeedsRefresh = false;
      drawGraphScreen();
      return;
    }

    if (isInside(screenX, screenY, GBTN_PRESS_X, GBTN_Y, GBTN_W, GBTN_H)) {
      lastTouchAction = millis();
      currentGraphMode = GRAPH_PRESSURE;
      graphNeedsRefresh = false;
      drawGraphScreen();
      return;
    }

    if (isInside(screenX, screenY, GBTN_GAS_X, GBTN_Y, GBTN_W, GBTN_H)) {
      lastTouchAction = millis();
      currentGraphMode = GRAPH_GAS;
      graphNeedsRefresh = false;
      drawGraphScreen();
      return;
    }

    if (isInside(screenX, screenY, BACK_BTN_X, BACK_BTN_Y, BACK_BTN_W, BACK_BTN_H)) {
      lastTouchAction = millis();
      currentScreen = SCREEN_STATS;
      statsNeedsRefresh = false;
      drawStatsScreen();
      return;
    }
  }

  if (currentScreen == SCREEN_ALARMS) {
    if (isInside(screenX, screenY, BACK_BTN_X, BACK_BTN_Y, BACK_BTN_W, BACK_BTN_H)) {
      lastTouchAction = millis();
      currentScreen = SCREEN_STATS;
      statsNeedsRefresh = false;
      drawStatsScreen();
      return;
    }
  }
}

// ======================================================
// SENSOR UPDATE
// ======================================================

void updateSensorData() {
  if (!bme.performReading()) {
    Serial.println("Messung fehlgeschlagen!");

    if (currentScreen == SCREEN_MAIN) {
      disableTouch();
      display.fillRect(220, FOOTER_Y, 135, FOOTER_H, NAVY);
      display.setTextColor(RED, NAVY);
      display.setTextSize(2);
      display.setCursor(225, 282);
      display.print("Messfehler");
    }
    return;
  }

  currentTemp = bme.temperature + TEMP_OFFSET;
  currentHum = bme.humidity;
  currentPressure = (bme.pressure / 100.0) + PRESSURE_OFFSET;
  currentGas = bme.gas_resistance / 1000.0;
  hasSensorReading = true;

  sumTemp += currentTemp;
  sumHum += currentHum;
  sumPressure += currentPressure;
  sumGas += currentGas;
  statsCount++;

  if (FORCE_ALARM_TEST) {
    airBad = true;
  } else {
    airBad = currentGas < GAS_BAD_THRESHOLD;
  }

  if (airBad && !lastAirBadState) {
    saveAlarmToLog("Schlecht");
  }

  lastAirBadState = airBad;

  if (!airBad) {
    alarmMuted = false;
  }

  Serial.println("------ Umweltwerte ------");
  Serial.print("Temperatur kalibriert: "); Serial.print(currentTemp); Serial.println(" C");
  Serial.print("Feuchtigkeit: "); Serial.print(currentHum); Serial.println(" %");
  Serial.print("Druck: "); Serial.print(currentPressure); Serial.println(" hPa");
  Serial.print("Gas: "); Serial.print(currentGas); Serial.println(" KOhm");
  Serial.print("Luftqualitaet: ");
  if (currentGas >= GAS_MEDIUM_THRESHOLD) Serial.println("Gut");
  else if (currentGas >= GAS_BAD_THRESHOLD) Serial.println("Mittel");
  else Serial.println("Schlecht");
  Serial.print("Alarm aktiv: "); Serial.println(airBad ? "JA" : "NEIN");
  Serial.print("Alarm quittiert: "); Serial.println(alarmMuted ? "JA" : "NEIN");
  Serial.println("-------------------------");

  if (currentScreen == SCREEN_MAIN) {
    updateMainMeasurementDisplay();
  }
}

// ======================================================
// LOGGING TIMER
// ======================================================

void handleLogging() {
  if (!hasSensorReading) return;

  if (millis() - lastLogSave >= logInterval) {
    lastLogSave = millis();
    saveMeasurementToLog();
    pruneOldLogEntries();
    pruneOldAlarmEntries();
    statsNeedsRefresh = true;
    graphNeedsRefresh = true;
  }
}

// ======================================================
// SETUP
// ======================================================

void setup() {
  Serial.begin(115200);
  delay(1500);
  Serial.println("Starte Umweltstation mit Alarm-Protokoll...");

  SPI.begin(18, 19, 23);
  pinMode(TFT_CS, OUTPUT);
  pinMode(TOUCH_CS, OUTPUT);
  digitalWrite(TFT_CS, HIGH);
  digitalWrite(TOUCH_CS, HIGH);

  pinMode(BUZZER_PIN, OUTPUT);
  digitalWrite(BUZZER_PIN, LOW);
  pinMode(BUTTON_PIN, INPUT_PULLUP);

  Wire.begin(I2C_SDA, I2C_SCL);
  initStorage();

  disableTouch();
  display.init(320, 480, 0, 0, ST7796S_RGB);
  display.setRotation(DISPLAY_ROTATION);
  showStartupMessage("Starte...", WHITE);

  disableDisplay();
  touch.begin();
  touch.setRotation(TOUCH_ROTATION);

  if (!rtc.begin()) {
    showStartupMessage("RTC fehlt!", RED);
    Serial.println("RTC nicht gefunden!");
    while (1) delay(1000);
  }

  Serial.println("RTC gefunden!");

  if (rtc.lostPower()) {
    Serial.println("RTC Zeit verloren. Setze Kompilierzeit...");
    rtc.adjust(DateTime(F(__DATE__), F(__TIME__)));
  } else {
    Serial.println("RTC Zeit vorhanden. Kein Reset der Uhrzeit.");
  }

  Serial.println("Starte BME680...");

  if (!bme.begin(0x76)) {
    Serial.println("BME680 nicht bei 0x76 gefunden. Teste 0x77...");
    if (!bme.begin(0x77)) {
      showStartupMessage("BME680 fehlt!", RED);
      while (1) {
        Serial.println("BME680 fehlt weiterhin.");
        delay(1500);
      }
    }
  }

  Serial.println("BME680 erfolgreich initialisiert!");
  bme.setTemperatureOversampling(BME680_OS_8X);
  bme.setHumidityOversampling(BME680_OS_2X);
  bme.setPressureOversampling(BME680_OS_4X);
  bme.setIIRFilterSize(BME680_FILTER_SIZE_3);
  bme.setGasHeater(320, 150);

  currentScreen = SCREEN_MAIN;
  drawStaticUI();
  updateClock();
  updateFooterStatus();
  lastLogSave = millis();
  Serial.println("Setup fertig.");
}

// ======================================================
// LOOP
// ======================================================

void loop() {
  handleTouch();
  handleButton();
  handleBuzzer();

  if (millis() - lastClockUpdate >= clockUpdateInterval) {
    lastClockUpdate = millis();
    updateClock();
  }

  if (millis() - lastSensorUpdate >= sensorUpdateInterval) {
    lastSensorUpdate = millis();
    updateSensorData();
  }

  handleLogging();

  if (currentScreen == SCREEN_STATS && statsNeedsRefresh) {
    statsNeedsRefresh = false;
    drawStatsScreen();
  }

  if (currentScreen == SCREEN_GRAPH && graphNeedsRefresh) {
    graphNeedsRefresh = false;
    drawGraphScreen();
  }

  if (currentScreen == SCREEN_ALARMS && alarmLogNeedsRefresh) {
    alarmLogNeedsRefresh = false;
    drawAlarmLogScreen();
  }
}
