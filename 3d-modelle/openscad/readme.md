# OpenSCAD Modell

Dieser Ordner enthält die CAD-Datei des Gehäuses für die ESP32 Umweltmessstation.

## Datei

### esp32_umweltstation_case.scad

OpenSCAD-Modell des Gehäuses.

Das Gehäuse wurde speziell für die verwendeten Komponenten entwickelt:

- ESP32
- ST7796S 4 Zoll Display
- XPT2046 Touchscreen
- BME680 Sensor
- DS3231 RTC Modul
- Buzzer
- Taster

## Eigenschaften

- Frontausschnitt für das Display
- Befestigungspunkte für die Elektronik
- Öffnungen für Sensorbelüftung
- Öffnung für USB-Kabel
- Platz für RTC und Sensor
- Verschraubbares Gehäuse

## Software

Zur Bearbeitung der CAD-Datei wurde OpenSCAD verwendet.

https://openscad.org

## Hinweis

Die OpenSCAD-Datei stellt die Master-Datei dar und kann für zukünftige Anpassungen weiter bearbeitet werden.
